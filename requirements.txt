#   * JDK (Java 11+)  : set JAVA_HOME. Runs TLC (Layer 2) and PRISM (Layer 3).
#                       Tested on JDK 20 (dev) and Temurin 21 (CI).
#   * Python 3.10+    : Layer 1 analyzer + Layer 3 calibration (pip pkgs below).
#   * Node.js (LTS)   : only to REGENERATE the Word docs (_docgen/*.js use the
#                       `docx` npm package). Deps are pinned in
#                       TrivySupplyChain/_docgen/package-lock.json — run
#                       `npm ci` in that folder. Not needed to run the analysis.
#   * Git for Windows : supplies the MinGW runtime DLLs (libwinpthread-1.dll,
#                       etc.) that PRISM's native library (prism.dll / CUDD)
#                       loads on Windows for Layer 3.
#
# --- Bundled in the repo (NO install needed) ---
#   * TLA+ tools / TLC 2.19  ->  TrivySupplyChain/tools/tla2tools.jar
#   * PRISM 4.10.1           ->  TrivySupplyChain/layer3/prism/
#
# --- Optional ---
#   * LaTeX (tectonic, bundled in Trivy-USENIX-paper/_tec/, or any TeX engine):
#     only to recompile the paper — main.pdf is already committed.
#   * Anthropic API key + the `anthropic` SDK: only for the ASI-Evolve runner
#     (TrivySupplyChain/asi_evolve/run_evolve.py). Pinned separately in
#     TrivySupplyChain/asi_evolve/requirements.txt so the core reproduction
